# SRV-M1M Asahi Infrastructure Platform

> High-performance ARM infrastructure platform demonstrating enterprise-grade services deployment, monitoring, and management on Apple Silicon.

## Overview

This project showcases the deployment and management of a high-performance infrastructure server based on Apple Silicon (M1/M2) running Asahi Linux. The SRV-M1M serves as the central data and service hub for the lab environment, demonstrating professional infrastructure engineering capabilities including container orchestration, high-availability services, and enterprise-grade monitoring.

The platform leverages the exceptional performance and efficiency of Apple Silicon to provide a cost-effective yet powerful infrastructure solution suitable for modern enterprise workloads.

## Architecture

**ARM Infrastructure Platform Architecture**:
- **Platform**: Apple Silicon (M1/M2) with Asahi Linux
- **Virtualization**: Container runtime with orchestration capabilities
- **Storage**: High-speed NVMe storage with optimized configuration
- **Networking**: Multi-homed configuration with Tailscale VPN integration
- **Security**: Firewall configuration with network segmentation
- **Observability**: Full-stack monitoring with metrics collection

**Core Services**:
- **Forgejo**: Primary Git server with full CI/CD capabilities
- **PostgreSQL**: High-performance database service
- **Monitoring Stack**: Prometheus, Grafana, and alert management
- **Container Runtime**: Application deployment and management
- **VPN Gateway**: Tailscale for secure remote access
- **Backup Services**: Automated backup and recovery systems
